# AgentOps 🤖💰

**AI Agent Cost & Compliance Hub** — Track, optimize, and audit your AI API spending across all providers.

[![Open in Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/theandroidaiarchitect-dev/AgentOps)
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/theandroidaiarchitect-dev/AgentOps)

## ✨ Features

- 📊 **Dashboard** — Real-time cost overview with beautiful charts
- 📈 **Analytics** — Detailed breakdowns by provider, model, and time period
- ⚙️ **Settings** — API key management, budget alerts, team management
- 🔐 **Compliance-ready** — EU AI Act audit logs structure
- 🎨 **Dark mode** — Premium dark-first UI with smooth animations

## 🚀 Quick Start

```bash
# Install
npm install

# Run dev
npm run dev
```

## 🛠️ Stack

- Next.js 16 + TypeScript
- Tailwind CSS v4 + shadcn/ui
- Framer Motion
- Recharts
- Clerk Auth (ready to configure)

## 🌐 Deploy

1. Push to GitHub
2. Import to Vercel
3. Set environment variables:
   - `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
   - `CLERK_SECRET_KEY`
4. Deploy 🚀
